# Reporting a Vulnerability

If you've found a security vulnerability in Progressive Chat, please contact
the project author privately:

**Anton Maurer** — [GitHub: progressive-chat](https://github.com/progressive-chat)

Please do NOT open a public issue for security vulnerabilities.
We follow responsible disclosure: you report privately, we fix, then announce.

## Supported Versions

| Version | Supported |
|---------|-----------|
| Development (main branch) | ✅ Active development |
| Release builds | ❌ Not yet released |

This project is a fork of Element Android Classic. For vulnerabilities in
the inherited Kotlin/Java codebase, also refer to:
https://github.com/element-hq/element-android/security
